# Images Directory

This project uses high-quality royalty-free placeholder images from Unsplash CDN.
Images are loaded directly via URL in the HTML files with lazy loading enabled.

To use local images instead:
1. Download images from https://unsplash.com
2. Place them in this folder
3. Update the `src` attributes in the HTML files

Recommended image sizes:
- Hero background: 1920x1080
- Gallery thumbnails: 600x600
- Event cards: 600x400
- Testimonial avatars: 100x100
